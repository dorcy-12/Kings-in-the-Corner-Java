# Kings in the Corner

Kings in the Corner ist ein Kartenspiel für 2-4 Spieler, welches mit einem 52 Karten Deck gespielt wird.
Unsere Version des Spiels bietet auch die Möglichkeit gegen Bot Spieler zu spielen.


## Spiel starten
Es muss Java auf dem Rechner installiert sein. Ein Download ist über die offizielle Website möglich.

https://www.java.com/download/ie_manual.jsp

Um das Spiel zu starten sind folgende Schritte zu beachten:

1. Die Server.jar Datei mit folgendem Befehl:'java -jar "Dateipfad"' in der Konsole starten.
(zum Beispiel "java -jar C:\Users\Admin\Desktop\Server.jar")

2. Die Client.jar Datei mit folgendem Befehl:'java -jar "Dateipfad"' in der Konsole starten.
(zum Beispiel "java -jar C:\Users\Admin\Desktop\Client.jar")

##Programm mit IntelljIdea starten

1. Installieren Sie sich IntelljIdea. Ein Download ist über die offizielle Website möglich.

https://www.jetbrains.com/idea/

2.Starten Sie ein Projekt über VCS und geben Sie folgende URL ein: "https://sepgit.informatik.uni-kl.de/ss22/gruppe02.git" und drücken Sie auf clone 

3. Geben Sie Ihre Benutzerdaten ein 

4. Starten Sie zuerst den Server mit rechtsklick und dann "Run 'Server.main()'"

5. Starten Sie dann den Client mit rechtsklick und dann "Run 'Client.main()'"
